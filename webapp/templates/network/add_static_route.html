Help & Examples
Common Route Examples:
Default Route:
  • Destination: default
  • Gateway: 192.168.1.1
  • Interface: eth0
Network Route:
  • Destination: 10.0.0.0/8
  • Gateway: 192.168.1.10
  • Interface: eth1
Host Route:
  • Destination: 203.0.113.1/32
  • Gateway: 192.168.1.254
  • Interface: eth0
Route Metrics:

Lower metric values have higher priority. Common values:

  • 10-50: High priority routes
  • 100: Normal priority (default)
  • 200-500: Lower priority routes
  • 600+: Backup routes
Note: Routes will be applied immediately and made persistent across reboots.
